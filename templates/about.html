About LogLite

LogLite: A Lightweight Framework for Real-Time Anomaly Detection and Analysis of System Logs

Detecting and diagnosing system anomalies with AI-powered log analysis

LogLite is a research project focused on developing a lightweight, efficient framework for real-time detection and analysis of anomalies in system logs. By leveraging modern deep learning techniques and explainable AI, LogLite identifies potential issues and provides clear, actionable insights.

LogLite

Version 1.0

Key Features
Anomaly Detection

Automatic identification of anomalous system logs using DistilBERT, a state-of-the-art transformer model.

Explainable AI

Using SHAP (SHapley Additive exPlanations) to provide transparent explanations of model predictions.

Interactive Dashboard

Visualize system health and anomaly distributions with an intuitive, responsive interface.

Root Cause Analysis

Identification of potential causes behind detected anomalies and recommended actions.

Training Dataset

LogLite's model was trained on the BGL (Blue Gene/L) supercomputer logs dataset, which contains labeled system logs from a large-scale supercomputing system. This dataset includes a diverse range of system events, errors, and anomalies, providing an excellent foundation for training a robust log analysis model.

The model classifies logs into normal operation or various types of kernel anomalies including memory issues, I/O problems, database errors, and critical system failures.
